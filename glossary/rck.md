# RCK — Glossary

| Korean | English | Notes |
|--------|---------|-------|
| 평판조회 | reference check | 핵심 서비스. 온라인/오프라인/사전심사 3종 |
| 온라인 평판조회 | online reference check | 시스템을 통한 자동 평판 수집 |
| 오프라인 평판조회 | offline reference check | 수동/대면 평판 수집 |
| 사전심사 | pre-screening | 채용 전 사전 확인 조회 |
| 질문지 | questionnaire | 평판 작성 시 사용하는 질문 세트 |
| 질문 템플릿 | question template | 질문지의 재사용 가능한 양식 |
| 후보자 | candidate | 평판조회 대상자 |
| 평판 작성자 | reference writer | 후보자의 이전 동료/상사로서 평판 작성하는 사람 |
| 기업 | company | 평판조회를 요청하는 채용 기업 |
| 관리자 | admin | 플랫폼 운영 관리자 |
| 체크포인트 | checkpoint | Admin이 개입하는 검수/확인 단계 |
| 리커트 척도 | Likert scale | 설문 응답 형식 (1~5 또는 1~7 스케일) |
| 선택지 | option / choice | 객관식 질문의 응답 항목 |
| 후속 질문 | follow-up question | 특정 응답에 따라 추가로 표시되는 질문 |
| AI 요약 | AI summary | OpenAI 기반 평판 결과 자동 요약 |
| 강점 | strength | AI 요약 리포트 필드 |
| 개선점 | areas for improvement | AI 요약 리포트 필드 |
| 퇴사사유 | reason for resignation | AI 요약 리포트 필드 |
| 포인트 | point/credit | 평판조회 요청 시 사용하는 내부 화폐 |
| 보고서 | report | 평판조회 결과 리포트 |
| 잔금 | final payment | 프로젝트 완료 후 남은 대금 |
| 유지보수 | maintenance | 프로젝트 종료 후 1개월 무상 버그 수정 기간 |
