# Glossary: DSA

| Korean | English | Notes |
|--------|---------|-------|
| 기술지도 | Technical Guidance | 건설 현장 기술 지도/감독 |
| 기술지도위원 | Technical Advisor | 현장 기술지도 수행자 (보고서 작성/수정) |
| 조직 어드민 | Org Admin | 조직 내 유저 권한 관리 (1명) |
| 조직관리자 | Org Manager | 기술지도위원 권한 포함 + 관리 기능 (n명) |
| 외부사용자 | External User | 현장소장, 공무원 등 — 초대받은 프로젝트만 접근 |
| 기술지도 보고서 | Technical Guidance Report | 현장 방문 후 작성하는 보고서 |
| 운행일지 | Operation Log / Vehicle Log | 차량 운행 기록 |
| 운행 리포트 | Operation Report | 월간 운행 요약 |
| 계약현황 | Contract Dashboard | 기술지도 계약 현황 |
| 손익관리 | Revenue/Profit Management | 프로젝트별 수익성 관리 |
| 성과관리 | Performance Management | 기술지도위원 성과 평가 |
| 입찰관리 | Bid Management | 건설 입찰 관리 |
| 협력사 | Partner | 외부 협력업체 |
| 문서대장 | Document Registry | 문서 관리 대장 |
| 드라이브 | Drive | 파일 저장소 |
| 인세프 | INSEP | 기존 기술지도 관리 시스템 (교체 대상) |
| 현품표 | Product Label | — |
| 위험요인 | Risk Factor | 보고서 내 현장 위험 항목 |
| 개선사항 | Improvement Item | 보고서 내 개선 권고 사항 |
| 플랜 | Plan | SaaS 요금제 (Free/Standard/Pro/Enterprise) |
