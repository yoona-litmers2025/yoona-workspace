# W81 (Everytalk) — Glossary

| Korean | English | Notes |
|--------|---------|-------|
| 일방향 채팅 | one-way chat | 관리자→회원 단방향 메시지 |
| 그룹 채팅 | group chat | 다수 참여 채팅방 |
| 통화 | VoIP call | Sendbird 기반 음성 통화 |
| 푸시 알림 | push notification | FCM (Android) / APNs (iOS) |
| 읽음 처리 | read receipt | 메시지 읽음/안읽음 상태 |
| 안읽음 배지 | unread badge | 채팅방 안읽은 메시지 수 |
| 채팅 미리보기 | chat preview | 채팅 목록에서 마지막 메시지 표시 |
| 미디어 업로드 | media upload | 이미지/영상 전송 |
| 포맷 정규화 | format normalization | 다양한 영상 포맷을 MP4/H.264/AAC로 통일 |
| 트랜스코딩 | transcoding | 영상 변환 처리 |
| 관리자 앱 | admin app | 기업 관리자용 앱 |
| 회원 앱 | member app | 일반 사용자용 앱 |
| 초대 | invitation | 채팅방 초대 |
