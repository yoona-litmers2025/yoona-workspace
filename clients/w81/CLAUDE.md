# Client: W81 (Everytalk)

## Overview
- **고객사명**: W81
- **프로젝트명**: Everytalk
- **설명**: 실시간 채팅 & 통화 앱 (Admin app + Member app). 1:1/그룹/일방향 채팅, VoIP 통화, 푸시 알림, 미디어 업로드 기능 포함.
- **담당자**: 변지수 팀장님
- **Tech Stack**: React Native, Sendbird (채팅/통화), Firebase (FCM push), NHN (SMS 인증)

## Communication Style
- 연락이 잘 안 됨 — 응답 대기 시간이 길 수 있음, 팔로업 필요
- 이전 개발사에서 기능 구현 미흡으로 소송 진행 중인 상태에서 우리에게 넘어온 프로젝트
- 불안감과 불신이 매우 높음 — 사소한 버그에도 민감하게 반응
- 대표가 직접 찾아와서 환불/법적대응 언급한 적 있음
- 온도가 매우 낮은 상태 — 모든 커뮤니케이션에서 신뢰 회복이 최우선
- 약속한 일정은 반드시 지켜야 함 — 한 번 더 어기면 관계 회복 불가능할 수 있음

## Tone
- 기본: 합니다체 (격식체) — 해요체보다 더 공식적으로
- 이슈 보고 시: 문제 + 원인 분석 + 대응 방안 + 일정을 반드시 함께 전달
- 절대 "확인해보겠습니다"로만 끝내지 않음 — 구체적 액션과 기한 필수
- 진행 상황을 선제적으로 공유 — 물어보기 전에 먼저 알려야 신뢰 회복

## Domain Context
- **비즈니스 도메인**: B2B 커뮤니케이션 앱 (기업 내부 채팅/통화)
- **앱 구성**: Admin app (관리자) + Member app (일반 사용자)
- **핵심 기능**: 채팅 (1:1/그룹/일방향), VoIP 통화, 푸시 알림, 미디어(이미지/영상) 전송
- **외부 연동**: Sendbird (채팅+통화), Firebase FCM (푸시), NHN (SMS)
