# Client: FreshMart

## Overview
- **고객사명**: FreshMart (프레시마트)
- **프로젝트명**: FreshMart
- **설명**: 신선식품 새벽배송 스타트업의 모바일 앱 리뉴얼. 기존 웹 기반 주문 시스템을 모바일 앱으로 전환하면서 주문/배송/회원 시스템 전면 개편.
- **담당자**: 박지현 대리

## Communication Style
- 꼼꼼하고 질문이 많으심 → 답변 시 근거와 이유를 함께 설명하면 좋음
- 답변은 빠른 편 → 질문 던지면 빠르게 회신 옴
- IT 스타트업 출신 → 기술 용어 어느 정도 이해하시지만 개발 깊이는 모르심
- 경쟁사 사례를 자주 언급하시며 비교 요청하심 → "다른 앱은 이렇던데" 식의 요청이 많음

## Tone
- 해요체 기본, 공식 문서는 합쇼체
- 노션 문서는 간결체 (도입/마무리만 합니다체)
- 전문적이면서도 친근한 톤 — 스타트업 분위기에 맞게

## Domain Context
- 신선식품 새벽배송 (fresh food dawn delivery)
- 수도권 대상 서비스
- 주요 프로세스: 상품 등록 → 주문 → 결제 → 배송 배정 → 새벽배송 → 수령 확인
- 컷오프 타임(밤 11시) 기반 익일 배송
- 콜드체인(저온 유통) 물류 체계
- 픽업존(무인 수령 장소) 배송 옵션
- 정기구독 모델 운영 (주 1회/2회/매일)
- 회원 등급제 + 적립금 시스템
